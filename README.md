# Smartfleet

Este projeto foi criado para aprimorar os processos de uma empresa de locação de veículos, solucionando ineficiências operacionais nos cadastros de automóveis, clientes e locações. Os objetivos específicos incluem a centralização e automação desses processos, melhoria da experiência do cliente e implementação de um sistema eficiente. Foram modelados três processos principais: o cadastro de veículos, o cadastro de clientes e a realização de locações.
