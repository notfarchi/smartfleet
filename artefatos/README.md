# Artefatos do Projeto

Este diretório contém os artefatos produzidos para o projeto SmartFleet. Eles foram organizados em subpastas para facilitar a navegação e identificação dos conteúdos.

## Estrutura do Diretório

- **/processos**: Contém os modelos de processos de negócio do sistema, como:
  - **cadastro_veiculos.bpmn**: Processo que descreve o fluxo de cadastro, edição e exclusão de veículos.
  - **cadastro_clientes.bpmn**: Processo que detalha o fluxo de cadastro, edição e exclusão de clientes.
  - **locacao.bpmn**: Processo que representa o fluxo de realização de locações de veículos.

- **/dados**: Inclui os modelos de dados utilizados no sistema, como:
  - **DER (Diagrama Entidade-Relacionamento)**: Representação gráfica das entidades e seus relacionamentos.
  - **Modelo Relacional**: Estrutura detalhada das tabelas e relações no banco de dados.
